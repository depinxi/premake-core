Moves the console cursor left by a number of columns.

```lua
term.moveLeft(columns)
```

### Parameters ###
`columns` is the number of columns to move left.

### Return Value ###
None

### Example ###
``` lua
term.moveLeft(2)
```

### See Also ###
* [term.clearToEndOfLine](term.clearToEndOfLine.md)

### Availability ###

Premake 5.0.0 beta 3 or later.
