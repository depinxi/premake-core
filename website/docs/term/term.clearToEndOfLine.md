Clears the console from the cursor location to the end of the line.

```lua
term.clearToEndOfLine()
```

### Parameters ###
None

### Return Value ###
None

### Example ###
``` lua
term.clearToEndOfLine()
```

Premake 5.0.0 beta 3 or later.
